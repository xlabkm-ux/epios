# Ключевые Роли и Персоны (User Personas & RBAC)

Epistemic OS использует ролевую модель доступа (RBAC), которая глубоко интегрирована как в API (Security Middleware), так и в UI (`SecurityContext`). Ниже описаны основные продуктовые роли.

## 1. Author (Автор артефакта / Contributor)
**Цель:** Описать техническое решение, доказать его обоснованность и получить "Approve" от экспертов.
- **Права:** Может создавать Workspace, загружать исходники (Sources), инициировать асинхронный MCP-маппинг. Может предлагать патчи (`ProposePatch`) для исправления замечаний.
- **Ограничения:** В строгих профилях оценки не может сам аппрувить свои же высокорисковые патчи.
- **UI Опыт:** Видит подсказки от AI (Overbroad claims, недостаток Evidence). Работает в основном с `GovernancePanel` (Propose) и `MissionPanel`.

## 2. Reviewer (Архитектор / Ревьюер / Approver)
**Цель:** Объективно оценить предложенное решение, выявить риски и слепые зоны, принять финальное решение.
- **Права:** Обладает правом решающего голоса (`CastVote`). Может изменять рейтинг доверия к источнику (`RatingPanel`), что напрямую влияет на статус готовности.
- **UI Опыт:** Основной фокус на `ReadinessPanel` (анализ блокеров) и `GraphCanvas` (визуальный поиск оторванных от доказательств утверждений). Кнопки "Approve" / "Reject" активны.

## 3. Observer (Наблюдатель / Viewer)
**Цель:** Изучить контекст принятого решения, понять историю (Traceability), не внося изменений.
- **Права:** Read-only доступ к графу, панелям и истории патчей.
- **Ограничения:** Отключены функции мутации графа. Запрещено голосование и создание патчей. Не видит контент, помеченный как `redacted` (в зависимости от настроек конфиденциальности).
- **UI Опыт:** Кнопки "Vote" и "Propose" задизейблены или скрыты. Фокус на функции экспорта и таймлайне аудита.

## 4. Admin (Администратор домена)
**Цель:** Настройка политик, шаблонов и управление жизненным циклом данных.
- **Права:** Управление `Template Library`, назначение ролей, форсированное удаление рабочих пространств (`Hard Delete`).
- **UI Опыт:** Работает в будущих `RBAC Admin UI` и `Template Selector` конфигураторах.
