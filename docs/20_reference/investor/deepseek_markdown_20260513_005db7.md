# Инвестиционное предложение: EpiOS

## Операционная система для управляемого, аудитируемого и безопасного ИИ в корпорациях

> **Суть одной фразой:** EpiOS превращает «чёрный ящик» генеративного ИИ в прозрачный процесс с контролем человека, неизменяемым следом для регуляторов и изоляцией данных.

---

## 1. Проблема: ИИ есть, доверия — нет

Корпорации массово внедряют AI-агентов (88% по данным McKinsey 2026).  
Но за рамки пилотов выходят только **33% проектов**.  
А чистую прибыль благодаря ИИ увеличивают **менее 5% компаний**.

**Корневая причина:** отсутствие доверия, аудита и управляемости.

Директора по комплаенсу и безопасности блокируют ИИ-агенты из-за трёх страхов:

1. **Непрозрачность решений** — «почему ИИ предложил именно это?»
2. **Невозможность ответить регулятору** — нет лога действий, причины изменений, цепи утверждений.
3. **Утечка данных** — модель получает доступ к сырым данным раньше, чем это разрешено политиками.

**Рыночное окно:** к 2028 году только на решения для AI governance придётся **$24 млрд**.

---

## 2. Решение: EpiOS — аудитируемый слой между ИИ и данными

EpiOS не является очередным чат-ботом или обёрткой вокруг GPT.  
Это **отдельный операционный слой**, который встраивается между любой LLM и корпоративными данными.

### Ключевые возможности

| Компонент | Что даёт клиенту |
| --- | --- |
| **Граф утверждений** | Каждый шаг ИИ логируется в защищённую цепочку (можно отдать аудитору). |
| **Human‑in‑the‑loop** | Критические действия не выполняются без явного одобрения человека. |
| **Изолятор данных (zero‑trust)** | Модель видит только те данные, которые прошли политики и маскирование. |
| **Агностицизм к моделям** | Работает с любыми LLM (OpenAI, Anthropic, Llama, Mistral) как в облаке, так и on‑prem. |
| **Режим «предлагаю — утверждаешь»** | Полный откат изменений, история версий, кто и почему одобрил. |

## 3. Почему сейчас — идеальный timing

| Фактор | Ситуация |
| --- | --- |
| **EU AI Act** | С 2026 года high-risk AI обязан иметь логи, человека в цикле и audit trail. Штрафы — до €30 млн. |
| **Gartner** | К 2028 году более 50% организаций перейдут на zero‑trust data governance. |
| **Agentic AI boom** | 67% компаний тестируют агентов, но только 14% внедрили approval workflow (Gravitee). |
| **Корпоративные бюджеты** | 59% ИТ-бюджетов идёт на AI, из них 24% — чисто на governance и риск (WalkMe). |

**Рынок созрел:** проблема стала критической, а готовых платформ класса «аудит + контроль + изоляция» почти нет.

## 4. Продукт: что уже работает

**Текущий статус:** закрытая альфа / pilot-ready.

- **MVP реализован:** граф утверждений, создание миссий, привязка источников, предложение изменений в артефакт, одобрение/отклонение человеком.
- **2 пилотных проекта** с enterprise-клиентами (один — страховая компания, второй — производитель медицинского оборудования).
- **Технологическая основа:** PostgreSQL, графовая модель утверждений, отдельное ядро для логов, адаптеры к LLM.

**Ближайшие 3 месяца:** запуск закрытой беты ещё для 5 клиентов, сбор метрик по времени принятия решений и снижению инцидентов.

## 7. Конкуренты — матрица 2×2

| | **Только аудит / observability** | **Аудит + контроль + изоляция данных** |
| --- | --- | --- |
| **Привязаны к одному вендору (OpenAI, Azure)** | Arize, WhyLabs | Microsoft Purview (частично) |
| **Агностик к моделям / облакам** | Credo AI, Monitaur | **EpiOS** — единственный в этом квадранте |

**Вывод:** прямых конкурентов с полным набором (агностицизм + human‑in‑the‑loop + zero‑trust + след для регулятора) **нет**.

Косвенные конкуренты (LangSmith, собственные разработки корпораций) либо не имеют встроенного approval workflow, либо непомерно дороги в кастомизации.

## 13. Ключевой инвестиционный тезис

EpiOS — это **страховка для бизнеса**, который уже использует ИИ или только собирается это делать.  
Мы решаем проблему, из-за которой 70% AI-проектов буксуют: отсутствие контроля, прозрачности и audit trail.

Первая волна AI-продуктов дала компаниям **скорость генерации**.  
Следующая волна — которую мы оседлаем прямо сейчас — требует:

- доказательности
- управляемости
- соблюдения регуляторных требований
- возможности сказать регулятору: «вот почему ИИ принял такое решение»

EpiOS — это **инфраструктурный слой для ответственного ИИ** в enterprise.

Мы ищем **$800k**, чтобы за 12 месяцев превратить MVP в enterprise-продукт с первыми 10 платящими клиентами и ARR $1.5M.
