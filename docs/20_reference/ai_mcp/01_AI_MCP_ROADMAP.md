# AI & MCP Roadmap

## 🟡 Горизонт 1: Prompt Management
- [ ] Вынесение захардкоженных промптов из `mapping-processor.ts` в БД или конфигурационные файлы.
- [ ] Версионирование промптов (чтобы старые артефакты могли быть перепарсены старой версией промпта).

## 🔵 Горизонт 2: Pluggable MCP Architecture
- [ ] Разработка интерфейса реестра MCP-плагинов в Admin UI.
- [ ] Реализация динамического роутинга: шаблон "Security" отправляет текст на один MCP-сервер, шаблон "ADR" — на другой.

## 🟣 Горизонт 3: Continuous AI Evaluation
- [ ] Внедрение фреймворка для оценки качества извлечения сущностей (например, RAGAS или LangSmith).
- [ ] Дашборд качества: процент успешных маппингов, время ответа LLM, стоимость токенов.
