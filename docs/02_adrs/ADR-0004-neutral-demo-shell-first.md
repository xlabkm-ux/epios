# ADR-0004 — Use Neutral Demo Shell as First Product Shell

## Status
Accepted

## Context
Starting with ChatAVG v3.0 as the primary shell would pull in legacy branding, complex workflows, and existing product assumptions. We need a clean way to prove the new platform primitives.

## Decision
The first product shell for Epistemic OS will be a **Neutral Demo Shell**, not branded as ChatAVG.

## Consequences
- **Positive**: Proves platform primitives without legacy weight; clearer open-source positioning; reusable for diverse demos and validation.
- **Negative**: ChatAVG v3.0 product shell is delayed; product identity is initially less market-specific.

## Revisit Trigger
Revisit after MVP if a branded product shell (like ChatAVG v3.0) becomes the primary priority for the next phase.
