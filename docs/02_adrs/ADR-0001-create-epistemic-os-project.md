# ADR-0001 — Create Epistemic OS Project

## Status
Accepted

## Context
There is a need for a new platform focused on traceable reasoning, evidence-backed artifacts, and human-in-the-loop decisions, separate from the ChatAVG v2.3 codebase.

## Decision
Create the Epistemic OS (epos) project as an independent open-source repository using a layered hexagonal architecture.

## Consequences
- New repository bootstrap required.
- Clear separation from ChatAVG legacy.
- Clean-slate for architectural principles.
