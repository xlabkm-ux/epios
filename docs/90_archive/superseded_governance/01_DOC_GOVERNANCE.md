# Регламент управления документацией (Continuous Concept Evolution)

**Документ:** Манифест жизненного цикла концепций и документации (DocFlow).
**Статус:** Утвержден.
**Цель:** Исключить расхождение между архитектурной мыслью (документацией) и реальным кодом (имплементацией), а также устранить смысловое дублирование.

## 1. Фундаментальный принцип: Код первичен, Документация — следствие
В проекте Epistemic OS документация не является неизменной догмой. Если в процессе реализации задачи (спринта) найдено более оптимальное архитектурное решение, **разработчик (и ИИ-агент) обязан написать оптимальный код**, даже если он противоречит первоначальной концепции.

## 2. Фазы жизненного цикла (Спринт-цикл)

Каждая итерация разработки обязана подчиняться следующему процессу:

### Фаза A: Exploration (Исследование)
- Перед написанием кода изучаются соответствующие файлы в `docs/...`.
- Цель: понять ожидаемую концепцию.

### Фаза B: Implementation (Имплементация)
- Написание кода, тестов, отладка.
- Допускаются любые изменения первоначального плана для достижения стабильности, производительности или лучшего UX.

### Фаза C: Post-Sprint Audit (Аудит и Дедупликация)
- Перед финальным коммитом проводится обязательная сверка.
- **Анализ:** Что было сделано иначе, чем планировалось?
- **Дедупликация:** Если новая реализация перекрывает несколько старых roadmap-ов (например, безопасность UI и RBAC API слились в один контекст), дублирующиеся концепции объединяются в один "источник правды".

### Фаза D: Concept Refinement (Развитие концепции)
- Обновление `00_..._INDEX.md` и `01_..._CONCEPT.md` в папке `docs/`.
- Выполненные пункты из `ROADMAP.md` помечаются `[x]`, либо удаляются.
- В следующий горизонт планирования добавляются новые задачи, вытекающие из свежего практического опыта.
- Только после этого выполняется `git commit` и `git push`.

## 3. Обязанности ИИ-агента
Агент Antigravity запрограммирован автоматически запускать Фазы C и D при завершении каждого смыслового блока работы. Пользователю не требуется напоминать об этом.
