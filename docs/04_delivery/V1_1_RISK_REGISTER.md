# V1.1 Risk Register

Risks for the 16-sprint plan.

## Key Risks
- **Scope Creep**: Expanding the ADR Review into a universal platform.
- **MCP Bypass**: Direct mutation via MCP bypassing strict domain controls.
- **Score Laundering**: Over-reliance on AI-generated correctness scores without human review.
- **Governance Overload**: Processes becoming too bureaucratic for effective delivery.
