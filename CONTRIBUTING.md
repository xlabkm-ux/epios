# Contributing to Epistemic OS

We welcome contributions! As an open-source project from day one, we value transparency and collaboration.

## Engineering Process

- **Domain-First**: Always start with the domain model.
- **Contract-First**: Define interfaces and schemas before implementation.
- **Traceability**: Ensure all changes are traceable and observable.

## How to Contribute

1. Check the [Project Roadmap](work_doc/epios_03_mvp_scope_and_6_week_roadmap.md).
2. Look for open issues or RFCs.
3. Submit a Pull Request with clear descriptions and tests.
